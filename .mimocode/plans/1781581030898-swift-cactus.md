# 入口页"系统设置"卡片状态显示改进

## 目标
将"系统设置"卡片中的硬编码"正常"改为基于 WebSocket 连接状态动态显示，带颜色圆点。

## 修改文件
- `garment-web/src/views/EntryHome.vue`

## 实现方案

### 1. 引入 WebSocket
在 `EntryHome.vue` 的 `<script setup>` 中导入 `useWebSocket` composable，获取 `connected` 和 `onlineUsers`。

### 2. 修改 cardStats.config
```javascript
// 修改前
config: [
  { label: '状态', value: () => '正常', unit: '' },
  { label: '在线', value: () => 1, unit: '人' },
],

// 修改后
config: [
  { label: '状态', value: () => connected.value ? '服务正常' : '服务异常', unit: '', isStatus: true },
  { label: '在线', value: () => onlineUsers.value.length || 1, unit: '人' },
],
```

### 3. 修改模板中的卡片 footer 渲染
在 `.cf-item` 中，当 `s.isStatus` 为 true 时，渲染一个带颜色的小圆点：
```html
<span class="cf-val">
  <span class="status-dot" :class="connected ? 'online' : 'offline'"></span>
  {{ fmtNum(s.value()) }}<small>{{ s.unit }}</small>
</span>
```

### 4. 添加样式
```css
.status-dot {
  display: inline-block;
  width: 8px; height: 8px;
  border-radius: 50%;
  margin-right: 6px;
  vertical-align: middle;
}
.status-dot.online { background: #22c55e; }
.status-dot.offline { background: #ef4444; }
```

## 验证
1. 启动前后端 (`cd garment-server && node server.js` + `cd garment-web && npm run dev`)
2. 访问入口页，确认"系统设置"卡片状态区域显示绿色圆点 + "服务正常"
3. 停止后端，确认状态变为红色圆点 + "服务异常"
4. 重启后端，确认状态自动恢复为"服务正常"
