## 阅读建议

1. **新人入职**: 先看 SOP-00(总览)→ 再看你岗位对应的 SOP-01~06
2. **忘记操作**: 直接翻你岗位 SOP 的目录,定位到具体视图
3. **异常情况**: 各 SOP 末尾有「常见问题」章节
4. **图文渲染**: 流程图采用业内通用的「Mermaid」语法编写,需要支持 Mermaid 渲染的 Markdown 编辑器查看(例如:VSCode、Typora、飞书文档、语雀等)。如发现流程图显示为代码,说明当前查看工具不支持,请升级或更换工具
