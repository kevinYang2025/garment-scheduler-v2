## 一、目视化排程甘特图（重点模块）

### 功能完善
| 功能 | 说明 |
|------|------|
| 日期范围 | 默认显示今天前1周 ~ 后3周，支持左右箭头滚动 |
| 今日高亮 | 当天日期列紫色高亮显示 |
| 拖拽排班 | 从左侧待排班列表拖拽到产线完成排班 |
| 拖拽移动 | 已排任务可拖拽到其他产线，自动重新计算日期 |
| 取消排班 | 双击任务条取消排班，同步更新 main_plan |
| 自动计算日期 | 排班时根据产线最后任务+日产量自动计算开始/结束日期 |
| 空产线默认 | 无排程的产线从明天开始排 |
| 休息日显示 | 与工作日历保持一致，不再仅按周末判断 |

### 布局优化
| 优化项 | 说明 |
|--------|------|
| 左右对齐 | 采用单一滚动容器+sticky定位，左右完全同步滚动 |
| 行高统一 | 左侧产线标签与右侧甘特条行高统一为56px |
| 车间标题对齐 | 左侧车间标题行与右侧空行一一对应 |
| 任务条裁剪 | 超出日期范围的甘特条自动裁剪不显示 |
| 左侧固定 | 横向滚动时左侧产线列固定不动 |

### 筛选增强
- 新增"全部类别"下拉筛选（按服装类别筛选产线）
- 搜索框同时支持搜班组名和服装类别
- 清除按钮一键清除所有筛选条件
